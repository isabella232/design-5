# PGA Artwork

### Public Git Archive Dataset

The dataset contains 3TB of repositories from GitHub ready to download. This includes all of the contents (git metadata and file contents) for all of the repositories on GitHub with 50 or more stars. The list of repositories was obtained from the [GHTorrent project](http://ghtorrent.org/), specifically from the snapshot of January 1st 2018. GHTorrent provides information about GitHub repositories but unfortunately doesnâ€™t include the file contents.

You can download it from the releases page at [github.com/src-d/datasets](github.com/src-d/datasets).
